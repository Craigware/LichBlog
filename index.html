Simple entity system

I have three different types of entities currently, dynamic entities, static entities, and etherial entities.
Dynamic entites encompass everything in the game that can move in a sporadic pattern. These are things like the player, bosses, and npcs.
Static entities are entities that don't tend to move. Examples of this would be houses, folliage, and points of interest.
Etherial entities are entities that can be interacted with but do not interact with the combat system at all. These are things like teleportation locations and item pickups.

Player Movement

The player utilizes a finite state machine in order to control actions. The state machine has 6 states, IDLE, MOVING, ROLLING, ATTACKING, CHANNELING, and STUNNED. Currently the only states that are implemented are IDLE, MOVING, and ROLLING.
When the player begins to move their movement speed ramps up over time, and when they stop it decreases. I believe that movement is the defining factor in a lot of different games, and that honing this part will be integral to how fun the game will be.
The player also has the option to roll, forcing them in a direction at an increased velocity. Right now the player cannot influence the roll as it is being performed, I don't know if this will be the case further in development.

Inventory

The inventory system is pretty simple, currently im using an array that contains a class that has a reference to an item and a count. Later down the line when I allow the player to sort their inventories themselfs using an array here instead of a dictionary is going to make things simpler.

Primitive Combat System

I decided to use static library class and enum for using weapons. If the player performs and attack and a weapon is currently selected in their inventory, it calls Attacks.UseWeapon(weapon, caster) which figures out how the weapons hitbox should be calculated based off a WEAPON_TYPE enum, if there is a hit then Attacks.UseEffect(weapon, entity) is called with the hit entity using a WEAPON_EFFECT enum to decide what effect the weapon has, both use a bunch of switch cases BB.
